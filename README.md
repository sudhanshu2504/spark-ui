## 🚀 How to Use Spark UI in Your Project

### Prerequisites

Make sure your project has:

- **React** (v18+)
- **Tailwind CSS** configured
- **Motion** (Framer Motion) - `npm install motion`

### Quick Start

1. **Browse Components**  
   Visit [sparkui.me/components](https://sparkui.me/components) to explore available components

2. **Copy the Code**  
   Click on any component to view its code, then copy it to your clipboard

3. **Paste into Your Project**  
   Create a new component file in your project and paste the code

4. **Customize**  
   Modify colors, spacing, and styles using Tailwind classes to match your design

### Example: Adding a Hero Section

```tsx
// 1. Visit sparkui.me/components/hero-parallax
// 2. Copy the component code
// 3. Create a new file in your project

// components/Hero.tsx
import { HeroParallax } from "@/components/ui/hero-parallax";

export default function Hero() {
  return <HeroParallax components={yourData} />;
}

// 4. Import and use in your page
import Hero from "@/components/Hero";

export default function Home() {
  return <Hero />;
}
```

### Installation (For Development)

If you want to run the Spark UI showcase website locally:

```bash
# Clone the repository
git clone https://github.com/sudhanshu2504/spark-ui.git
cd spark-ui

# Install dependencies
npm install

# Run the development server
npm run dev

# Open http://localhost:3000
```
